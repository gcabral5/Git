Instructions

Concentration, also known as Match Match, Memory, Pelmanism, Shinkei-suijaku, Pexeso or Pairs, is a card game in which all of the cards are laid facedown on a flat surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards.

Why should I play?

Studies have shown that simple card games like this actually help improve memory retention over a period of time.
